# DRYscope Starter Kit

Your own restoration estimating app — free, no code, ~15 minutes.

**Start here → [`prompts/00-START-HERE.md`](prompts/00-START-HERE.md)**

The short version: open this folder in a Claude Cowork session and run the three prompts in order — (1) company + sales tax, (2) pricing (your own rates, or derived from open data with [ODAPM](https://getodapm.github.io)), (3) optional free web deploy for your crew.

Every price ships at zero on purpose. Your numbers should be yours.
