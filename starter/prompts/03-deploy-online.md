# Step 03 — Put it online (optional, free)

This gives your crew a web link that opens on any phone — no app store, no server, no cost. You'll use GitHub (free account) and its drag-and-drop upload: **no terminal, no keys.**

1. Create a free account at **github.com** (or sign in).
2. Go to **github.com/new** → Repository name: `estimator` (anything works) → **Public** → Create repository.
3. On the new repo's page, click **"uploading an existing file."**
4. In your file manager, open your finished `starter` folder, select **everything inside it**, and drag it onto the upload area. (Mac: press ⌘⇧. first so hidden files show.)
5. Commit message: `my estimator` → **Commit changes**.
6. Repo **Settings → Pages** → Source: *Deploy from a branch* → Branch **main**, folder **/ (root)** → Save.
7. A minute later your estimator is live at `https://YOUR-USERNAME.github.io/estimator/` — bookmark it on every crew phone.

**Updating later:** repeat the upload (Add file → Upload files) — same-named files are replaced. Your saved quotes live on each device, not the website, so updates never touch them.

**Heads-up:** a public page means your prices are visible to anyone with the link. Most owners are fine with that (transparency is the point); if you're not, GitHub offers private repos + Pages on paid plans.
